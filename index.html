Where do I get the Spreadsheet ID?

  1. Open this spreadsheet

  2. File -> Make a copy

  3. Fill the spreadsheet with real data. It should look like this:

  4. Next step is make this file Public; File -> Publish to the web -> Publish

  5. Extract the Spreadsheet ID from the URL, for example if your URL is this one: https://docs.google.com/spreadsheets/d/13J7bCeY26AaUGgW_2gOyLH5aulZFKQM_zwxMhfUJTGQ/pubhtml the id is 13J7bCeY26AaUGgW_2gOyLH5aulZFKQM_zwxMhfUJTGQ

  6. That's all, now you have it, place it inside the "Spreadsheet ID" input file
